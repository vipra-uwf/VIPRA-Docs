{\ttfamily Types} are declared in the {\ttfamily Types Declaration} of a {\ttfamily Behavior}.

A {\ttfamily Types Declaration} is what says which types are being used in the {\ttfamily Behavior}.

{\ttfamily Behaviors} can only have 1 {\ttfamily Types Declaration} which can include up to 64 {\ttfamily Types}.

{\ttfamily Types Declarations} are slightly different from other declarations, in that they do not have components; instead each type is simply listed out.

\hypertarget{md_docs_2pages_2researchers_2behaviors_2behaviors__syntax_autotoc_md227}{}\doxysubsection{\texorpdfstring{Groups}{Groups}}\label{md_docs_2pages_2researchers_2behaviors_2behaviors__syntax_autotoc_md227}
For most users, {\ttfamily Groups} and {\ttfamily Types} refer to the same thing.

The main difference between a {\ttfamily Type} and a {\ttfamily Group} is that there is a base {\ttfamily Group} for each {\ttfamily Behavior}, being \textquotesingle{}Pedestrian\textquotesingle{} (or \textquotesingle{}Pedestrians\textquotesingle{}).

Each pedestrian with a {\ttfamily Type} is considered in that {\ttfamily Type}\textquotesingle{}s {\ttfamily Group}.

Pedestrians with {\ttfamily Composite Types} are in a {\ttfamily Group} for each {\ttfamily Type}.

{\ttfamily Selectors} are how pedestrians are selected for a certain Type.

Selecting pedestrians is done through a {\ttfamily Selector} declaration. The basic syntax is as follows\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{Selector:}
\DoxyCodeLine{\ \ Type:\ <types>}
\DoxyCodeLine{\ \ From:\ <type>\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ //\ Optional,\ defaults\ to\ base\ pedestrians\ group}
\DoxyCodeLine{\ \ Select:\ <selection\_criteria>}
\DoxyCodeLine{.}

\end{DoxyCode}



\begin{DoxyItemize}
\item \texorpdfstring{$\ast$}{*}\+Type\texorpdfstring{$\ast$}{*} -\/ The Type to assign to selected pedestrians.
\item \texorpdfstring{$\ast$}{*}\+From\texorpdfstring{$\ast$}{*} -\/ The Group to select pedestrians from.
\item \texorpdfstring{$\ast$}{*}\+Select\texorpdfstring{$\ast$}{*} -\/ The exact {\ttfamily Select} criteria to use. Available {\ttfamily Select} criteria are listed under Available Select Criteria.
\end{DoxyItemize}

Selectors are applied with precedence equal to the order they appear in the {\ttfamily Behavior} file.
