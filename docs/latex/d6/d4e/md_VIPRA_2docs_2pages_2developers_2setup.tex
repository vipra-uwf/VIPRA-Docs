VIPRA can either be used\+:
\begin{DoxyItemize}
\item as a Library for use in other projects
\item as a Standalone Simulation
\end{DoxyItemize}\hypertarget{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md542}{}\doxysubsection{\texorpdfstring{Requirements}{Requirements}}\label{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md542}
The following are requirement for running the VIPRA simulation.


\begin{DoxyEnumerate}
\item \href{https://cmake.org/download/}{\texttt{ CMake}} version 3.\+14 or newer.
\item A compiler that supports C++20.
\end{DoxyEnumerate}

Additionally for Parameter Sweeps\+:
\begin{DoxyEnumerate}
\item MPI, either \href{https://www.open-mpi.org/software/ompi/v5.0/}{\texttt{ open\+MPI}} or \href{https://www.mpich.org/}{\texttt{ mpich}}
\end{DoxyEnumerate}

The following are required for development of the Behaviors DSL.


\begin{DoxyEnumerate}
\item ANTLR4 -\/ version 4.\+13.\+1 or newer.
\end{DoxyEnumerate}

\hypertarget{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md546}{}\doxysection{\texorpdfstring{Standalone Simulation}{Standalone Simulation}}\label{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md546}
A Make\+File is provided with the proper command to build a standalone simulation from the provided {\ttfamily main.\+cpp} in the top level directory.

To build the simulation, in the parent directory run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make}

\end{DoxyCode}


Alternatively, run one of the following\+:


\begin{DoxyItemize}
\item {\ttfamily make release} for compiling in release mode.
\item {\ttfamily make debug} for compiling in debug mode.
\item {\ttfamily make perf} for compiling the performance testing version.
\end{DoxyItemize}\hypertarget{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md547}{}\doxysubsection{\texorpdfstring{Using MPI}{Using MPI}}\label{md_VIPRA_2docs_2pages_2developers_2setup_autotoc_md547}
{\ttfamily MPI} is enabled with the addition of {\ttfamily mpi=on} to the make command

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ release\ mpi=on}

\end{DoxyCode}


\begin{quote}
!!! IMPORTANT NOTE\+: make sure that the version of MPI you are compiling with matches the version you run the simulation with. \end{quote}
