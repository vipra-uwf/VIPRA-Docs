Parameters are loaded into each module before they are initizalized. The exact steps of a simulation run are shown in \href{../developers/overview.md\#simulation-steps}{\texttt{ {\ttfamily Simluation Steps}}}.

Parameters can use any {\ttfamily Para\+Reader} module, the \doxysectlink{md_docs_2pages_2researchers_2base__modules}{{\ttfamily JSON}}{0} input module provided works for this.

\hypertarget{md_docs_2pages_2researchers_2parameters_autotoc_md361}{}\doxysection{\texorpdfstring{Parameter Sweeps}{Parameter Sweeps}}\label{md_docs_2pages_2researchers_2parameters_autotoc_md361}
Parameter sweeps run many simulations with randomized parameters to get a better idea of the possible outcomes.\hypertarget{md_docs_2pages_2researchers_2parameters_autotoc_md362}{}\doxysection{\texorpdfstring{Parameters}{Parameters}}\label{md_docs_2pages_2researchers_2parameters_autotoc_md362}
Parameters are provided as they are with a normal run, with some extra rules.

{\ttfamily NOTE\+:} parameters are randomized in a non-\/parameter sweep run if provided in this format\hypertarget{md_docs_2pages_2researchers_2parameters_autotoc_md363}{}\doxysubsection{\texorpdfstring{Non-\/\+Random Values}{Non-\/\+Random Values}}\label{md_docs_2pages_2researchers_2parameters_autotoc_md363}
Any parameter that should not be randomized is provided as it is normally.


\begin{DoxyCode}{0}
\DoxyCodeLine{single\_string:\ "{}example1"{}}
\DoxyCodeLine{single\_float:\ 1.0}

\end{DoxyCode}


Array\textquotesingle{}s can still be used as they normally would be.


\begin{DoxyCode}{0}
\DoxyCodeLine{array\_param:\ [1.0,\ 2.0,\ 3.0]}

\end{DoxyCode}
\hypertarget{md_docs_2pages_2researchers_2parameters_autotoc_md364}{}\doxysubsection{\texorpdfstring{Discrete Random Values}{Discrete Random Values}}\label{md_docs_2pages_2researchers_2parameters_autotoc_md364}
Parameters can be randomized between a set of discrete values.

For each simulation run, the actual value for the parameter is randomly chosen from the list given with a uniform distribution.

Examples\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{discrete\_string\_param:\ [\ "{}option1"{},\ "{}option2"{},\ "{}option3"{}\ ]}
\DoxyCodeLine{discrete\_float\_param:\ [\ 1.0,\ 1.2,\ 1.4\ ]}

\end{DoxyCode}


{\ttfamily NOTE\+:} Array parameters can NOT be randomized


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ will\ NOT\ work}
\DoxyCodeLine{random\_array\_param:\ [\ [1,2],\ [3,4]\ ]}

\end{DoxyCode}
\hypertarget{md_docs_2pages_2researchers_2parameters_autotoc_md365}{}\doxysubsection{\texorpdfstring{Continuous Random Values}{Continuous Random Values}}\label{md_docs_2pages_2researchers_2parameters_autotoc_md365}
Numerical parameters can also be randomized within a continuous range.

For each simulation run, the actual value for the parameter is randomly chosen with a uniform distribution from {\ttfamily min} to {\ttfamily max} inclusively.

Examples\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{continuous\_param:\ \{}
\DoxyCodeLine{\ \ "{}min"{}:\ 1.0,}
\DoxyCodeLine{\ \ "{}max"{}:\ 4.0}
\DoxyCodeLine{\}}

\end{DoxyCode}
