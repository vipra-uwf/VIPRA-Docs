VIPRA Behaviors is a DSL for creating human behaviors for use in pedestrian dynamics simulations. Its goal is to provide an easy way for researchers without programming experience to test how human behaviors may affect pedestrian dynamics.

{\ttfamily Behaviors} are a way to describe {\ttfamily Actions} that pedestrians take under a certain {\ttfamily Condition} without the use of a traditional programming language.

{\ttfamily Behaviors} are comprised of four major parts\+:
\begin{DoxyEnumerate}
\item Types
\item Selectors
\item Events
\item Actions
\end{DoxyEnumerate}
